Hammerspoon
mac-自定义小工具🔧
使用步骤
1.	安装 hammerspoon
2.	copy 至 ~/.hammerspoon/中
3.	reload config



#   先有功能
```
require "Spoons/window-management" -- 窗口管理
require "Spoons/clipboard" -- 粘贴板
require "Spoons/ime" -- 设置各种应用的默认输入法
require "Spoons/volume" -- 控制声音
require "Spoons/wi" -- wifi切换
require "Spoons/weather" -- 天气 todo
```
